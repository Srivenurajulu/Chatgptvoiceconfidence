# Voice Search Journey - Wireframes

This document contains generated UI wireframes corresponding to the Voice Search Journey flow defined in the architecture.

## 1. Entry & Nudge
**Goal:** Introduce the next-gen voice search features with a simple tooltip nudge on the landing screen to build awareness before the first interaction.

## 2. Voice Capture & STT Processing
**Goal:** Show the intermediate "Breaking Down" processing state while keeping the transcribed snippet visible for the user to review. This ensures the user feels in control of the draft text before it is sent.
