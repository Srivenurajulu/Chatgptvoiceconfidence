# Voice Confidence Mode

Welcome to the Voice Confidence integration repository! This project serves as an architectural blueprint and React Web implementation of a "Speak-to-Draft" interface designed to make AI voice prompting more socially discreet, accurate, and comfortable for professionals in shared spaces or open offices.

## 📁 Repository Structure

*   **`Architecture.md`**: The complete system design, broken down into the Client Application, API flow, Data Retention policies, and the Phase-by-Phase Plan.
*   **`SetupGuide.md`**: The technical documentation describing the environment dependencies, component setups, and how to execute this repository.
*   **`prompts.md`**: Pre-engineered LLM developer system prompts allowing you to rapidly scaffold the code phases via AI workflows in the future.
*   **`Wireframes.md`**: Low-fidelity UI visual references for the Contextual Nudges, Processing States, and Rewrite suggestion modals.
*   **`voice-confidence-app/`**: The unified, master React module utilizing `expo-web` to render the entire Voice Confidence implementation stack dynamically!
*   **`Phase[1-4].../`**: The isolated developmental phase folders highlighting the architectural separation of concerns before they were merged into the main application wrapper.

## 🚀 Getting Started
To spin up and interact with the unified visual web app locally in your browser, refer entirely to the [Setup Guide](./SetupGuide.md) file!
